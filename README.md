# Final Project

## Description
This is a simple Android application that allows you to manage a list of strings. You can add, remove and display the strings in the list.

## Features
- Add a string to the list.
- Add a string to the list multiple times.
- Remove a string from the list.
- Remove a string from the list multiple times.
- Display the content of the list.
- The number of times a string is added or removed is limited to 5.

## How to use
1.  Enter a text in the "Introduce un texto" field.
2.  Enter a number in the "Introduce un número" field.
3.  Click the "Añadir" button to add the text to the list.
4.  Click the "Eliminar" button to remove the text from the list.
5.  Click the "Mostrar" button to display the content of the list.
